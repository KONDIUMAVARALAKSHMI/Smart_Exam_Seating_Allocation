# 🎓 Smart Exam Seating Allocation System

[![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://www.python.org/downloads/)
[![Django](https://img.shields.io/badge/Django-4.2-green.svg)](https://www.djangoproject.com/)

**Project Title:** NEXT GENERATION APPROACH FOR EXAM HALL DISTRIBUTION THROUGH UNIQUE AND ANAMOLUS SEMANTIC TECHNIQUES (B.Tech Final Year Project)

## 📖 Overview

The traditional manual exam seating arrangement process is often prone to errors, time-consuming, and lacks efficiency. Challenges such as data inaccuracies, cumbersome seat allocation, and manual communication pose significant hurdles for educational institutions. 

To address these issues, the **Smart Exam Seating Allocation System** provides an automated, dynamic, and error-free approach to exam hall distribution. By leveraging modern web technologies and specific semantic techniques, the system automatically allocates students to exam halls with specific configurations (e.g., 2 or 3 students per bench), avoiding duplication and streamlining the administration workflow.

## 🚀 Features

### 👨‍💻 Admin Panel
* **Bulk Data Import:** Import student information seamlessly using Excel (`.xlsx`) files. 
* **Dynamic Exam Hall Setup:** Configure exam rooms on the fly with customizable fields for number of rooms, total benches, total seats, and students per bench (2 or 3 choices).
* **Timetable & Faculty Management:** Add and manage exam timetables. Add faculty profiles, assign invigilation duties, and view/delete records.
* **Automated Seat Allotment:** Automatically generate fair and unique seat allocations for students based on the configured exam halls.
* **Comprehensive Reporting:** 
    * Generate **Excel Reports** for complete seat allotments.
    * Generate **Department-Wise PDF Reports**.
    * Generate **Room-Wise PDF Reports**.
    * Download **Attendance Sheets**.

### 👩‍🏫 Staff (Faculty) Panel
* **Secure Login:** Authenticated access for staff members.
* **Profile Management:** View and manage personal profiles and reset passwords.
* **Invigilation Duties:** View assigned exam invigilation schedules seamlessly.

### 🎓 Student Panel
* **Personal Portal:** Students can log in to view their specific exam details.
* **Seat Allotment Check:** View exact allotted Room Number, Bench Number, and Seat Number for their upcoming exams.
* **Profile Management:** Update passwords and view basic profile data.

## 🛠️ Technology Stack

* **Backend:** Python, Django Framework
* **Database:** SQLite3 (Default, easily switchable to MySQL/PostgreSQL)
* **Frontend:** HTML, CSS, JavaScript, Django Templates (with `django-widget-tweaks`)
* **Libraries:** `pandas` (for manipulating Excel files), PDF generation libraries (for report generation).

## ⚙️ Installation and Setup

Follow these instructions to set up the project on your local machine:

**1. Clone the repository**
```bash
git clone https://github.com/KONDIUMAVARALAKSHMI/Smart_Exam_Seating_Allocation.git
cd Smart_Exam_Seating_Allocation/in-house-main
```

**2. Create a virtual environment**
```bash
python -m venv venv
```

**3. Activate the virtual environment**
* **Windows:**
  ```bash
  venv\Scripts\activate
  ```
* **macOS/Linux:**
  ```bash
  source venv/bin/activate
  ```

**4. Install dependencies**
```bash
pip install -r requirements.txt
```
*(If `requirements.txt` is missing, ensure you install `django`, `django-widget-tweaks`, `pandas`, and your PDF generation library e.g., `xhtml2pdf`)*

**5. Apply Database Migrations**
```bash
cd EXAMSEATINGAUTOGENERATEDSYSTEM
python manage.py makemigrations
python manage.py migrate
```

**6. Create Superuser (Admin account)**
```bash
python manage.py createsuperuser
```

**7. Run the Local Development Server**
```bash
python manage.py runserver
```

**8. Access the Application**
Open your web browser and navigate to `http://127.0.0.1:8000/`.

## 📁 Directory Structure
* `adminapp/` - Contains logic and models for admin interactions, seat allotment, and report generation.
* `staff/` - Contains functionality for staff members to check their invigilation schedule.
* `student/` - Contains functionality for students to check their exam hall seating.
* `EXAMSEATINGAUTOGENERATEDSYSTEM/` - Main Django configuration folder (`settings.py`, `urls.py`).
* `templates/` - HTML templates for the frontend.
* `static/` - Static assets (CSS, JS, Images).

## 📜 Downloadable Reports Detailed

1. **Excel Report:** A complete system-wide seat allotment report generated in `.xlsx` format for institutional records.
2. **Department Wise PDF Report:** Segregated PDF report containing allotment details filtered by academic department.
3. **Room Wise PDF Report:** Formatted PDF displaying a visual representation of the allocated students per room, making it easier for invigilators to manage.
